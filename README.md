# TankLevelControl

Este proyecto permite monitorear en tiempo real el nivel de tanques de 10.000 litros mediante un ESP32 y un sensor ultrasónico.

Tecnologías: ESP32, C++, ASP.NET Core MVC, Entity Framework Core, WiFi y MQTT/HTTP.

Cómo correr la web: Abrir el proyecto en Visual Studio, configurar la base de datos y ejecutar con IIS Express.

Cómo cargar firmware: Compilar y cargar el código en el ESP32 desde Arduino IDE o PlatformIO.

Estado del proyecto: En fase inicial de arquitectura y estructura básica.
